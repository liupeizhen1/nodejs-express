##  新增文档

#### 路径： http://localhost：9090/lpz/good

#### type : POST

#### 参数类型： json  两个参数：1.title(表示标题)   2.body(表示文档内容)  参数类型：string   都是必填的！！！ 


####  返回的数据为：
{
    "code":0 , (0代表成功，-1代表失败)
    "msg": ok ,
    "data":{
        "id"：xxxx,
        "title": xxxx,
        "body":xxxx
    }
}
